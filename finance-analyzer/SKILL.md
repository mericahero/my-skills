# Finance Analyzer

## 功能介绍

该skill用于金融分析和投资组合管理，支持以下功能：

- **财务分析**：分析财务报表、计算财务比率、评估公司财务健康状况
- **投资组合构建**：根据风险偏好和投资目标构建多元化投资组合
- **收益算法实现**：计算投资组合的预期收益、风险和绩效指标
- **风险评估**：评估投资组合的风险水平、波动率和最大回撤
- **资产配置优化**：基于现代投资组合理论(MPT)优化资产配置
- **金融建模**：构建DCF模型、敏感性分析和情景分析

## 何时调用

当用户需要：
- 分析公司财务报表和财务比率
- 构建符合特定风险偏好的投资组合
- 计算投资组合的预期收益和风险
- 优化资产配置以实现风险调整后收益最大化
- 进行金融建模和估值分析
- 评估投资策略的绩效
- 分析市场数据和行业趋势

## 使用指南

### 基本使用步骤

1. **明确分析目标**：清晰说明您的财务分析或投资组合构建目标
2. **提供相关数据**：提供财务数据、市场数据或投资标的信息
3. **设置参数**：指定风险偏好、投资期限、目标收益等参数
4. **分析结果**：接收详细的分析报告和投资组合建议
5. **优化调整**：根据需要调整参数并重新分析

### 详细功能说明

#### 财务分析功能
- 分析资产负债表、利润表和现金流量表
- 计算关键财务比率（流动性、盈利能力、运营效率、偿债能力）
- 进行趋势分析和同业比较
- 评估公司财务健康状况和增长潜力

#### 投资组合构建功能
- 基于风险偏好构建多元化投资组合
- 支持股票、债券、ETF、商品等多种资产类别
- 考虑资产间相关性以优化分散化效果
- 提供多种投资组合构建策略（等权重、市值加权、风险平价等）

#### 收益算法功能
- 计算预期收益率、波动率和夏普比率
- 实现CAPM、Fama-French三因子模型等收益模型
- 进行蒙特卡洛模拟预测投资组合表现
- 计算风险调整后收益指标

#### 风险评估功能
- 评估投资组合的系统性风险和非系统性风险
- 计算VaR (Value at Risk) 和CVaR
- 分析投资组合的最大回撤和恢复周期
- 进行压力测试和情景分析

#### 资产配置优化功能
- 基于现代投资组合理论构建有效前沿
- 优化资产配置以实现特定风险下的收益最大化
- 考虑交易成本和税收影响
- 提供动态资产配置策略

## 示例场景

**用户请求**：
```

```

**处理步骤**：
1. 收集苹果公司2024年的财务报表数据
2. 计算流动性比率、盈利能力比率、运营效率比率和偿债能力比率
3. 与行业平均水平和历史数据进行比较
4. 分析公司财务趋势和健康状况
5. 提供详细的财务分析报告

**用户请求**：
```
请为风险中性的投资者构建一个包含美国股票、债券和国际资产的多元化投资组合，目标年化收益率为8-10%。
```

**处理步骤**：
1. 确定投资组合的资产类别和权重范围
2. 收集各类资产的历史收益和风险数据
3. 基于现代投资组合理论优化资产配置
4. 计算最优投资组合的预期收益、风险和夏普比率
5. 提供投资组合构建方案和建议

### 示例3：收益算法实现

**用户请求**：
```
请实现一个投资组合收益计算算法，考虑资产间相关性，并计算不同市场情景下的表现。
```

**处理步骤**：
1. 设计投资组合收益计算模型
2. 收集资产的历史收益率和相关性数据
3. 实现收益计算算法，考虑资产权重和相关性
4. 进行不同市场情景的模拟分析
5. 提供算法实现和分析结果

## 技术实现

### 核心技术

- 使用Python的金融库（如pandas、numpy、scipy）进行数据处理和分析
- 实现现代投资组合理论(MPT)算法进行资产配置优化
- 使用统计方法和机器学习模型预测资产收益
- 构建蒙特卡洛模拟模型评估投资组合风险
- 实现财务比率计算和财务报表分析算法

### 关键算法

1. **有效前沿计算**：使用二次规划或启发式算法计算有效前沿
2. **风险评估**：使用历史模拟法、参数法或蒙特卡洛模拟计算VaR
3. **资产配置优化**：基于风险调整后收益最大化的优化算法
4. **收益预测**：使用时间序列分析和因子模型预测资产收益
5. **投资组合绩效评估**：计算Sharpe比率、Sortino比率、信息比率等绩效指标

### 数据处理

- 处理财务报表数据和市场历史数据
- 清洗和标准化金融数据
- 构建金融时间序列数据库
- 处理缺失数据和异常值

## 注意事项

1. **数据质量**：分析结果的准确性依赖于输入数据的质量和完整性
2. **模型假设**：金融模型基于特定假设，实际结果可能有所不同
3. **市场风险**：投资存在市场风险，过去表现不代表未来结果
4. **计算复杂性**：某些算法可能需要较长的计算时间
5. **参数敏感性**：模型结果可能对输入参数敏感
