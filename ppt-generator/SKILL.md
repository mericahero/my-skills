# PPT生成器

## 功能介绍

PPT生成器是一个强大的技能，能够：

1. **根据用户需求生成PPT**：根据用户提供的主题、内容要点等信息，自动创建结构清晰、内容丰富的PPT文档
2. **文档转化为PPT**：支持将多种类型的文档转化为PPT格式，包括：
   - 图片文件
   - HTML网页
   - PDF文档
   - 文本文件
   - Markdown文档

## 使用场景

当用户需要以下服务时，应调用此技能：

- 创建新的PPT演示文稿
- 将现有文档转换为PPT格式
- 为会议、演讲准备幻灯片
- 基于资料自动生成演示内容

## 使用方法

### 方法一：直接生成PPT

用户可以提供以下信息，系统将根据这些信息生成PPT：

1. 主题（必填）：PPT的核心主题
2. 内容要点：需要包含的主要内容
3. 目标受众：演示的目标人群
4. 风格要求：希望的PPT风格（如商务、学术、创意等）
5. 页数要求：期望的幻灯片数量

### 方法二：文档转PPT

用户可以提供以下类型的文档，系统将其转换为PPT：

1. **图片文件**：将图片按顺序排列为PPT，可选择是否添加标题和描述
2. **HTML网页**：提取网页中的主要内容和图片，组织为PPT
3. **PDF文档**：提取PDF中的文本和图片，转换为结构化的PPT
4. **文本文件**：根据文本内容自动分段，生成PPT
5. **Markdown文档**：根据Markdown格式自动生成对应结构的PPT

## 示例

### 示例1：直接生成PPT

用户输入：
```
创建一个关于人工智能发展趋势的PPT，目标受众是企业管理层，风格要专业商务，包含以下要点：1. AI技术现状 2. 行业应用案例 3. 未来发展预测 4. 企业应对策略，共10页左右。
```

系统将生成一个符合要求的PPT演示文稿。

### 示例2：文档转PPT

用户输入：
```
将这份PDF年报转换为PPT，重点突出财务数据和业务亮点。
```

系统将提取PDF中的关键信息，转换为结构清晰的PPT。

## 技术原理

1. **内容分析**：分析用户提供的信息或文档内容，提取关键要点
2. **结构规划**：根据内容自动生成合理的PPT结构
3. **内容组织**：将信息组织为幻灯片形式，确保逻辑清晰
4. **格式转换**：对于不同类型的文档，使用相应的转换技术提取内容
5. **PPT生成**：根据分析结果生成最终的PPT文档

## 注意事项

1. 对于大型文档转换，可能需要一定处理时间
2. 转换结果可能需要用户进行适当调整和优化
3. 对于图片质量较低的文档，转换后可能影响PPT效果
4. 系统会自动保留原始文档中的重要信息，但可能会进行合理的概括和重组
